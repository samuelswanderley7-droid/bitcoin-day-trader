# Bitcoin Day Trader

## Project Documentation

### Overview
Bitcoin Day Trader is a trading bot that executes trades with the aim of profiting from fluctuations in the price of Bitcoin. The bot uses algorithmic trading strategies to identify the best times to buy and sell.

### Features
- Real-time trading capabilities
- Enrollment of trading strategies
- Performance tracking and analytics

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/samuelswanderley7-droid/bitcoin-day-trader.git
   ```
2. Navigate to the project directory:
   ```bash
   cd bitcoin-day-trader
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

### Usage
- Run the bot using:
   ```bash
   npm start
   ```
